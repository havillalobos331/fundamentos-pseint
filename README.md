### Antes de Empezar
Este material asume que usted esta utilizando una computadora (de escritorio o laptop) capaz de correr PSeInt.
Para instalar PSeInt:

1. Ir a la [siguiente liga](https://pseint.sourceforge.net/index.php?page=descargas.php)
2. Seleccionar la version de PSeInt correcta para el sistema operativo que esta utilizando (Windows, MacOS, o Linux).
3. Seguir las instrucciones de instalación proveidas por el programa descargado.
